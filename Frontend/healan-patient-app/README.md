# Healan Patient App

اپ موبایل بیمار با تم بنفش — ورود OTP پورتال.

## قابلیت‌ها

- خانه با اسلایدر سایت + میانبرها
- سوابق ویزیت/رزرو
- ثبت و جدول فشار خون
- یادآور داروها
- چت‌بات RAG
- رزرو نوبت
- پروفایل

## اجرا

```bash
cd Frontend/healan-patient-app
npm install
npm run web
```

پورت پیش‌فرض وب: `8083`
